## 📊Malaysia Income & Expenditure Dashboard (Power BI) ##

### 📌 Overview 

This project presents an interactive Power BI dashboard analyzing household income, expenditure, and access to amenities across Malaysian states. The dashboard provides insights into regional disparities and living standards.

--- 

### 🎯 Objective

- To analyze income and expenditure patterns across states
   
- To compare living standards and access to basic amenities
   
- To identify regional disparities using data visualization
---

### 📊 Dashboard Features

- Multi-page dashboard including:

  i. Income analysis
  
  ii. Expenditure analysis
  
  iii. Amenities comparison
  
- Interactive visuals for state-level comparison
  
- Card indicators to highlight key insights

  ---

 ### 📈 Key Insights
 
- Most states show high access to basic amenities (>99%)

- Sabah and Kelantan exhibit relatively lower access levels
  
- Income and expenditure patterns vary significantly across regions
  
- Higher income states tend to have higher expenditure levels  

---

### 🛠 Tools Used

- Power BI (Dashboard Development & Data Visualization)
   
- Excel (Data Preparation)
